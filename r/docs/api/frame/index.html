<Frame />

Component

Frame defines the editable area in your page editor. It is rendered based on the editor's internal state (i.e. Nodes).

const { connectors, setProp, ...collected } = useNode(collector);

#Reference

#Props

  • json?String
    The serialised Nodes
  • childrenReact.ReactElement<Canvas>
    This would define how the editor state would look like by default when there is nothing passed to the `json` prop. Must begin with a <Canvas /> which creates the Root Node.

#Examples

#The default screen

import {Editor, Frame, Canvas} from "cradt";
const App = () => {
  return (
    <div>
      <h2>My App!</h2>
      <Editor>
        <h2>My Page Editor</h2>
        <Frame> 
          <Canvas is={Container}> // defines the Root Node
            <h2>Drag me around</h2>
            <MyComp text="You can drag me around too" />
            <Canvas is="div" style={{background: "#333" }}>
              <p>Same here</p>
            </Canvas>
          </Canvas>
        </Frame>
      </Editor>
    </div>
  )
}
